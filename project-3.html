IOT Automatic Plant watering system

A system that estimates and measures the moisture of soil in order to operate an irrigation system, restoring water as need

Code Link

Project Overview

Objective- * To design an automatic irrigation system thereby saving time & power for the farmer. * To design a proto-type and test it on small scale so that we can create an effective solution for the problems on large scale. * Eliminating human interaction as much as possible in the process. * Providing the plants with water only when they require. * Providing a more modular system for easy upgrades. * Easy monitoring from anywhere in the world.
